CSS (Cascading Style Sheets).

Es un lenguaje que permite darle estilo a un documento HTML.

Ejemplo sin CSS (Solo HTML):

<h1>Bienvenido a mi Talento Tech.</h1>
<p>Esto es un parrafo.</p>

Expliacion:
El codigo anterior, genera un texto simple en negro sobre un fondo blanco.


Ejemplo con CSS:

h1{
  color:blue;
  text-align:center;
}

El titulo <h1> se vuelve azul y centrado.

1. APLICAR CSS A UNA PAGINA WEB.

Existen 3 forma de aplicar CSS:

1.1. CSS en linea (Inline CSS).

CCS se aplica directamente dentro de la etiqueta HTML mediante el atributo style.

<p style="color: red;"> Este texto es de color rojo.</p>

Nota: Lo anterior funciona, pero se vuelve dificil de mantener en proyectos grandes.


1.2. CSS Interno (Internal CSS).

Se escribe dentro de la etiqueta <style> en al seccion <head> del documento HTML.

<head>
  <style>
    p{ color: blue; }
  </style>
</head>

Nota: Util solo para pruebas rapidas, pero no para grandes proyectos.


1.3. CSS Externo (External CSS).

El CSS se aguarda en un archivo separado con extension .css y se enlaza en el documento HTML. Para usar un archivo externo, se deben seguir los siguientes pasos:

  • Paso 1: Crear el archivo styles.css
body {
  background-color: ligthgray;
  font-family: Arial, sans-serif;
}
  • Paso 2: Enlazar el archivo CSS en el archivo HTML.
<link rel="stylesheet" href="styles.css">

Nota: Separa el diseño del contenido y facilita la administracion del sitio web.

1.5. Regla CSS.

Una regla es una sentencia que describe como se debe presentar un elemento HTML en una pagina web. Cada regla consta de una selector y un bloque de declaraciones, que definen las propiedades y sus valores correspondientes para elementos seleccionados.

Estructura de una regla:

  • Selector: indica a que elementos de HTML se le aplicara la regla. Puede ser un nombre de etiqueta, una clase, un Id o una combinacion de estos.

    • clase: se utiliza para aplicar estilos a multiples elementos que comparten caracteristticas similares. Puede ser reutilizada en un multiples elementos de la pagina.

    • Id: se utiliza para identificar un elemento unico en la pagina. Debe ser unico en la pagina, no se puede repetir.

  • Llaves {}: delimitan el bloque de declaraciones de la regla.

  • Declaraciones: especifican las propiedades CSS y sus valores.

    • Propiedad: indica que aspectos del elemento se quiere modificar, por ejemplo: color, font-size, margin, etc.

    • Valor: define como se vera la propiedad, por ejemplo: red, 16px, 10px, etc.

  • Punto y coma ;: separa cada declaracion dentro del bloque de declaraciones.

2. SELECTORES CSS.

Existen varios grupos de lectores, entre estos se destacan:

2.1. Selectores Simples.

El grupo de selectores mas sencillo se orienta a elementos HTML, ademas de clases, IDs y otros atributos que se pueden agregar a una etiqueta HTML.

2.1.1. Selector Universal.

Un selector universal, tambien conocido como comodin, coincide con cualquier elemento. Por ejemplo:

* {
  color: hotpink;
}

La regla anterior hace que cada elemento HTML de la pagina tenga el texto en color rosa intenso.



2.1.2. Selectores Tipo.

Un selector tipo hace coincidir un elemento HTML directamente. Por ejemplo:

section {
  padding: 2em;
}

La regla anterior hace que cada elemento <section> tenga 2em de padding en todos lados.



2.1.3. Selectores de Clase.

Un elemento HTML puede tener uno o mas elementos definidos en su atributo class. El selector clase hace coincidir cualquier elemento al que se le haya aplicado esa clase. Por ejemplo:

Codigo HTML

<div class="mi-clase"></div>
<p class="mi-clase"></p>
<button class="mi-clase"></button>

Cualquier etiqueta a la que se le aplique la clase tendra el texto de color rojo:

Codigo CSS

.mi-clase {
  color: red;
}

Campista, observe que el . solo esta presente en el CSS y no en el HTML. Esto se debe a que el caracter . le indica al lenguaje CSS que coincida con los miembros del atributo de clase.



2.1.4. Selector ID.

Un elemento HTML con atributo id debe ser el unico elemento de una pagina con ese valor de ID. Se selecciona elementos con un selector ID de las siguiente manera:

# prueba {
  border: 1px solid blue;
  color: orange;
}

La regla anterior, aplicara un borde color azul y un color de texto naranja al elemento que tiene un id de prueba, de la siguiente manera:

<div id="prueba"></div>

Al igual que con el selector de clase ., se usa un caracter #para indicarle a CSS que busque un elemento que coincida con el id que lo sigue, para el ejemplo, id=prueba.



2.1.5. Selector de Atributos.

Se puede buscar elementos que tengan un atributo HTML determinado o valor determinado para un atributo HTML con el selector de atributos. Para indicarle a CSS que busque atributos, se usa el selector con corchetes [ ]. Por ejemplo:

[data-type='primary'] {
  color: red;
}

La regla anterior busca todos los elementos que tienen un atributo data-type con un valor primary, de la siguiente manera:

<div data-type="primary"></div>

En lugar de buscar una valor especifico de data-type, tambien se puede buscar elementos con atributo presente, independientemente de su valor. Asi:

[data-type] {
  color: red;
}

Codigo HTML:

<div data-type="primary"></div>
<div data-type="secondary"></div>

Ambos elementos <div> tendran el texto en color rojo.

Para usar selectores de atributos que distinguen mayusculas de minusculas, se agrega un operarador s al selector de atributo.

[data-type='primary' s] {
  color: red;
}

Lo anterior significa que, si un elemento HTML tuviera un data-type de Primary, en luagar de primary, no tendria el texto en color rojo. Tambien se puede hacer lo opuesto (sin distincion de mayusculas y minusculas) con un operador i.

Ademas de los operadores de mayusculas y minusculas, tiene acceso a operadores que coinciden con partes de cadena dentro de los valores de atributos. Por ejemplo:

/*: Un href que contenga*/
[href*='ejemplo.com'] {
  color: red;
}

/*^: Un href que empiece con*/
[href^='https'] {
  color: green;
}

/*$: Un href que finalice con .com */
[href$='.com'] {
  color: blue;
}



2.1.6. Selectores de Agrupacion.

Un selector no tiene que coincidir con un solo elemento. Para agrupar varios selectores, se deben separar por comas ,. Por ejemplo:

strong, em, .mi-clase, [campista] {
  color: red;
}

En el ejemplo anterior, el cambio de color se extiende a los elementos <strong> y <em>. Tambien se extiende a una clase llamada mi-clase y un elemento que tiene un atributo campista

3. PROPIEDADES ESENCIALES CSS.

Las propiedades CSS permiten modificar la apariencia de los elementos HTML. Las mas importantes son:

3.1. Propiedad color (Color del texto).

La propiedad color define el color del texto dentro de un elemento.

Ejemplo:

CSS

p {
  color: red; /* color rojo usando el nombre */
}

h1 {
  color: #0000ff; /* color azul usando codigo hexadecimal */
}

h2 {
  color: rgb(0, 255, 0); /* color verde usando RGB */
}

h3 {
  color: hsl(240, 100%, 50%): /* color azul usando HSL (Tono, Saturacion y luminosidad) */
}



3.2. Propiedad background-color (color de fondo).

Esta propiedad define el color de fondo de un elemento.

Ejemplo:

CSS

body {
  background-color: lightgray;
}

h1 {
  background-color: yellow;
}

Se pueden usar los mismos formatos que en la propiedad color (nombre, hexadecimal, RGB y HSL).



3.3. Propiedad background-image (fondo con imagen).

body {
  background-image: ulr("ruta_imagen/nombre_imagen.jpg");
  background-repeat: no-repeat;
  background-size: cover;
  background-position: center;
  background-attachment: fixed;
}



3.4. Propiedad font-size (Tamaño del texto).

La propiedad fon-size define el tamaño de la fuente del texto.

Ejemplo:

CSS

p {
  font-size: 16px; /* Tamaño en pixeles */
}

h1 {
  font-size: 2em; /* Relativo al tamaño base (2 veces mas grande) */
}

h2 {
  font-size: 120%; /* Relativo al tamaño predeterminado */

Unidades de medida mas comunes:

  • Pixeles (px): tamaño fijo, por ejempo, 16px.

  • Ems (em): Relativo al tamaño del elemento padre, por ejemplo, 2em.

  • Porcentajes (%): Relativo al tamaño del contenedor, por ejemplo, 140%.

  • Rems (rem): Relativo al tamaño de la raiz (html), por ejemplo, 1.5rem.



3.5. Propiedad font-weight (Grosor del texto).

Esta propiedad controla el peso o grosor del texto.

Ejemplo:

CSS

p {
  font-weight: normal;
}

h1 {
  font-weight: bold;
}

h2 {
  font-weight: 300; /* Texto mas delgado */
}

h3 {
  font-weight: 900; /*Texto extra grueso */

Valores disponibles:

  • normal: valor por defecto.

  • bold: negrita.

  • lighter: mas delgado que normal.

  • bolder: mas grueso que normal.

  • Valores numericos: 100 (mas delgado) hasta 900 (mas grueso).



3.6. Propiedad text-align (Alineacion de texto).

Define la alineacion del texto dentro de un elemento.

Ejemplo:

p {
  text-align: left;
}

h1 {
  text-align: center;
}

h2 {
  text-align: justify;
}

h3 {
  text-align: right;
}

4. Modelo de Caja (Box Model).

Es el sistema que usa CSS para representar como se estructura y ocupa espacio un elemento HTML en una pagina web. Cada elemento se comporta como una "caja" compuesta por varias capas:

4.1. Partes del Modelo de Caja.

De adentro a hacia afuera, una caja esta compuesta por:

  • Content (contenido): area donde aparece el texto o imagen, etc.

  • Padding (relleno): espacio interno entre el contenido y el borde.

  • Borde (borde): borde que rodea el padding y el contenido.

  • Margin (margen): espacio externo entre el borde del elemento y otros elementos.

4.1.1. Propiedad margin (Margenes exteriores).

Define el espacio entre un elemento y los elementos vecinos.

Ejemplo:

CSS

p {
  margin: 20px; /* Margen de 20px en todos los lados */
}

h1 {
  margin: 10px 20px; /* 10px arriba y abajo, 20 px a los lados */
}

h2 {
  margin: 5px 10px 15px 20px; /* Arriba, derecha, abajo, izquierda */
}



4.1.2. Propiedad padding(Espacio interno).

Define el espacio dentro de un elemento, entre su contenido y su borde.

Ejemplo:

p {
  padding:20px; /* Espaciado interno de 20px en todos los lados */
}

h1 {
  padding: 10px 15px; /* 10px arriba/abajo, 15px izquierda/derecha */
}

h2 {
  padding: 5px 10px 15px 20px; /* Arriba, derecha, abajo, izquierda */
}



4.1.3. Propiedad border (bordes del elemento).

Define un border alrededor de un elemento.

Ejemplo:

p {
  border: 2px solid black; /* Borde negro solido de 2px */
}

h1 {
  border: 3px dashed red; /* Borde rojo punteado */
}

h2 {
  border: 4px double blue; /* Borde azul doble */
}

Valores disponibles:

  • Estilo:
    • solid: solido.
    • dashe: puenteado.
    • dotted: con puntos.
    • double: doble linea.

4. Transicciones en CSS.

Una transiccion permite que los cambios de propiedades ocurran de forma suave y no abrupta.

Sintaxis:

selector {
  transition: propiedad duracion funcion-temporizacion retardo;
}

4.1. Subpropiedades de la propiedad transition y valores disponibles:

Subpropiedad Valores disponibles Descripción
transition-property all, color, background, transform, etc. Qué propiedad CSS se va a animar.
transition-duration 0.5s, 500ms, 1s Tiempo que tarda la transición.
transition-timing-function ease, linear, ease-in, ease-out, ease-in-out, steps(n), cubic-bezier(...) Define la curva de velocidad.
transition-delay 0s, 1s, 200ms Tiempo de espera antes de iniciar la transición.



4.2. Funciones de tiempo en CSS.

Función Qué hace
linear El cambio ocurre a velocidad constante desde el inicio hasta el final.
ease Comienza lento, acelera en el medio y desacelera al final. (valor por defecto)
ease-in Comienza lento y luego acelera.
ease-out Comienza rápido y termina lento.
ease-in-out Comienza lento, acelera en el medio y termina lento.



Ejemplo: Uso de la propiedad transition.

<style>
  .boton {
    background: steelblue;
    color: white;
    padding: 12px 24px;
    border: none;
    border-radius: 8px;
    font-size: 16px;
    cursor: pointer;

  }

  .boton:hover {
    background: darkblue;
    transform: scale(1.05);
    color: yellow;
  }
</style>

5. Animaciones en CSS.

Una animacion permite cambiar propiedades CSS a lo largo del tiempo, en varias etapas, sin necesidad de interaccion del usuario.

5.1. Bloque @keyframes.

@keyframes es una regla css que define los fotogramas clave de una animacion. Cada fotograma representa un punto en el tiempo de la animacion con estilos especificos.

Cuando se usa @keyframes, el navegador interpola (calcula) los valores intermedios automaticamente para crear un efecto visual fluido.

5.1.1. Anatomia del bloque @keyframes.

@keyframes nombre_animacion {
  porcentaje1 {propiedad: valor;}
  porcentaje2 {propiedad: valor;}
  .
  .
  .
  porcentajen {propiedad: valor;}
}

Explicacion:

  • @keyframes: palabra clave obligatoria.
  • nombre_animacion: nombre que se usara con la propiedad animation-name.
  • porcentajes: representa el progreso de la animacion (de 0% a 100%).
  • Dentro de cada porcentaje, se define las propiedad CSS que se quiere que cambien.

Ejemplo: Aplicacion de animacion.

<style>
@keyframes mover {
  0%   { transform: translateX(0); }
  100% { transform: translateX(200px); }
}

.prueba {
  animation: mover 2s linear infinite alternate;
}
</style>

<div class="prueba">
  <p>Campista, esta es una prueba de animacion de movimiento horizontal.</p>
</div>

Explicacion del codigo anterior:

  • @keyframes mover: se define una animación llamada mover.

  • 0%: inicio de la animación.

    • translateX(0): el elemento comienza sin moverse horizontalmente.

  • 100%: fin de la animación.

    • translateX(200px): el elemento se mueve 200 píxeles hacia la derecha.

  • mover: nombre del bloque @keyframes.

  • 2s: duración de la animación.

  • linear: velocidad constante.

  • infinite: se repite para siempre.

  • alternate: va y regresa (de 0 → 100%, luego de 100% → 0%).



5.2. Propiedades comunes animables con @keyframes.

  • transform (translate, rotate, scale, skew).

  • opacity.

  • background-color.

  • color.

  • width, height.

  • left, top, right, bottom.

  • box-shadow.

❌ No animables:

display

position (aunque puedes animar top, left, etc.)

visibility (pero se puede simular con opacity)

6. Diseño Responsivo con Media Queries (@media).

El diseño responsivo permite que un sitio web se adapte automaticamente a diferentes tamaños de pantalla, como computadores, tablets, iPads y telefonos moviles. Para lograrlo en CSS, se utiliza Media Queries.

6.1. @media en CSS.

@media es una regla en CSS que permite definir estilos especificos segun ciertas condiciones, como:

  • El ancho de la pantalla: para adaptar diseños a moviles o computadores.

  • La orientacion: Horizontal o vertical.

  • El tipo de dispositivo: pantalla, impresora, TV, etc.

Ejemplo de una Media Query basica:

@media screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

Explicacion del codigo:

  • @media: Indica que se esta definiendo una Media Query.

  • screen: Aplica los estilos solo en pantallas (no en impresoras).

  • and (max-width: 600px): Se aplica si el ancho de la pantalla es de 600 pixeles o menos.

  • background-color: lightblue;: Cambia el color de fondo azul claro solo en pantallas pequeñas.

    Nota: Si la pantalla es mayor a 600px, el fondo mantiene su color original.



6.2. Funcionamiento de las Media Queries.

Las Media queries detectan el tamaño de la pantalla y aplican diferentes reglas CSS segun la condicion establecida.

Ejemplo de uso practico de Media Query:

/* Estilos generales (se aplican en todas las pantallas) */

body {
  font-size: 18px;
  color: black;
}

/* Si la pantalla es menor a 800px, el texto sera mas pequeño */

@media screen and (max-widht: 800px) {
  body {
    font-size: 16px;
    color: darkgray;
  }
}

/* Si la pantall es menor a 500px, el texto sera aun mas pequeño */

@media screen and (max-width: 500px) {
  body{
    font-size: 14px;
    color: red;
  }
}

Nota: El codigo anterior, permite que la pagina se vea bien en cualquier dispositivo.



6.3. Tipos de Media Queries.

Las Media Queries pueden aplicarse en diferentes situaciones. Las mas importantes son:

6.3.1. Por Ancho de Pantalla (width).

Permite aplicar estilos segun el ancho de la pantalla. Por ejemplo:


@media screen and (max-width: 768px) {
    body {
        background-color: lightgray;
    }
}



6.3.2. Por Altura de Pantalla (height).

@media screen and (max-height: 500px) {
    body {
        background-color: yellow;
    }
}

Nota: Util en pantalla con poca altura, como celulares en modo horizontal.



6.3.3. Por Orientacion (orientation).

Permite aplicar estilos segun si el dispositivo esta en modo vertical (portrait) o modo horizontal (landscape). Por ejemplo:



@media screen and (orientation: landscape) {
  body {
    background-color: green;
  }
}

Nota: Util para adaptar interfaces de videojuegos o formularios en celulares.



6.3.4. Para Impresoras (print).

Se puede definir estilos especificos cuando el usuario imprime la pagina.

Ejemplo:

@media print {
  body {
    font-size: 12px;
    color: black;
    background-color: white;
  }
}

Nota: Util para hacer que una web sea mas legible en papel.

8. Diseño flexible con Flexbox.

Flexbox (Flexible Box Layout) es un modelo de diseño en CSS que permite organizar elementos de manera eficiente, alineandolos horizontal o verticalmente sin necesidad de usar sintaxis complicadas.

Ventajas de usar Flexbox:

  • Facilita la alineacion y distribuccion de elementos.

  • Se adapta automaticamente al tamaño de la pantalla (diseño responsivo).

  • Permite controlar el espacio entre elementos sin usar margenes.

  • Funciona bien en cualquier navegador moderno.



8.1. Activacion de Flexbox en CSS.

Para activar Flexbox, se usa la propiedad:

display: flex;

Esto convierte un contenedor en un flex container y permite que sus elementos internos (hijos) sean flex-items.

Ejemplo basico:

HTML

<section class="contenedor">
  <div class="caja">1</div>
  <div class="caja">2</div>
  <div class="caja">3</div>  
</section>

CSS

.contenedor {
  display: flex;
  background-color: lightgray;
}

.caja {
  background-color: blue;
  color: white;
  padding: 20px;
  margin: 10px;
}

Con la combinacion de los codigos anterioes, se logra que los elementos cajase alienean automaticamente en una fila.



8.2. Propiedades de Flexbox.

Las propiedades esenciales son:

8.2.1. flex-direction: Direccion de los elementos.

Define los elementos en fila (row) o columna (column).

.contenedor {
  display: flex;
  flex-direction: row; /* Fila (predeterminada) */
}

Otras opciones disponibles:

  • row-reverse: Invierte el orden en fila.

  • column: Los elementos se apilan en columna.

  • column-reverse: Invierte el orden en la columna.



8.2.2. jutify-content: Alineacion horizontal.

Controla como se distribuyen los elementos en el eje principal.

.contenedor {
  display: flex;
  justify-content: center; /* Centra los elementos */
}

Opciones disponibles:

  • flex-start (predeterminado): Alinea los elementos a la izquierda.

  • flex-end: Alinea los elementos a la derecha.

  • center: Centra los elementos.

  • space-between: Distribuye los elementos con espacios entre ellos.

  • space-around: Agrega espacio a los lados de cada elemento.

  • space-evenly: Espacia los elementos de manera uniforme.

.contenedor {
  display: flex;
  justify-content: space-between;



8.2.3. align-items: Alineacion vertical.

Controla como se alinean los elementos en eje secundario (vertical si flex-direction: row).

Opciones disponibles:

  • flex-start: Alinea los elementos arriba.

  • flex-end: Alinea los elementos abajo.

  • center: Centra los elementos verticalmente.

  • stretch (predeterminado): Expande los elementos para ocupar el alto total del contenedor.

  • baseline: Alinea los elementos segun su texto base.

Ejemplo con stretch:

.contenedor {
  display: flex;
  align-items: stretch;
}

Los elementos se estiran verticalmente para llenar el contenedor.

8.3. Propiedad de los Flex Items (Elementos Hijos).

Las propiedades esenciales son:

8.3.1. flex-grow: Expandir elementos.

Determina que elementos pueden crecer para llenar el espacio disponible.

.caja {
  flex-grow: 1; /* Cada caja crecera de maner uniforme */
}

Ejemplo:

.caja:nth-child(2) {
  flex-grow: 2; /* La segunda caja creceera el doble que las otras */
}

El selector :nth-child(n) en CSS se usa para seleccionar elementos específicos dentro de un grupo de elementos hermanos.



8.3.2. flex-shrink: Reduccion de tamaño.

Controla cuanto se encoge un elemento cuando no hay suficiente espacio.

.caja {
  flex-shrink: 0; /* No permite que elemento se reduzca */
}

Si no se especifica, los elementos pueden encogerse automaticamente.



8.3.3. flex-basis: Tamaño inicial.

Define el tamaño inicial de un elemento antes de aplicar flex-grow o flex-shrink.

.caja {
  flex-basis: 200px; /* Caja caja empieza con 200px de ancho */
}



Ejemplo con diferentes valores:

.caja:nth-child(1) { flex-basis: 50px;}
.caja:nth-child(2) { flex-basis: 150px;}
.caja:nth-child(3) { flex-basis: 100px;}

Cada caja inicia con un tamaño diferente.

8.4. Imagenes Responsivas.

Una imagen responsiva es aquella que se adapta automaticamente al tamaño de la pantalla del dispositivo (celuar, tablet, computador), sin deformarse ni salirse del diseño.

Codigo clave:

img {
  max-width: 100%; /*La imagen no sobrepasa el ancho de su contenedor*/
  height: auto; /*Mantiene la proporcion original (relacion ancho-alto)*/
}

  • max-width: 100%
    Si la imagen es mas grande que su contenedor, se reducira. Si es mas pequeña, conservara su tamaño original.

  • height: auto
    Ajusta la altura automaticamente para no deformar la imagen.



Buenas practicas:

  • Coloque la imagen en un contenedor con ancho flexible (%,vw, etc).

  • Evite usar width o height fijos en px si desea que la imagen se adapte.

  • Use display: block; si ve espacios blanco bajo la imagen.

img {
  max-width: 100%;
  height: auto;
  display: block; /* Evita margen blanco por debajo */
}
